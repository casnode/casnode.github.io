L'environnement du tutoriel est Ubuntu 20.04

Installer Casnode sous le panneau BT Linux​

Préparation du travail​

Après avoir installé le panneau BT, le navigateur visite le panneau BT, sélectionne le magasin de logiciels, recherche et installe MySQL, puis recherche un noeud, vous pouvez voir qu'il y a un gestionnaire PM2, installez le gestionnaire PM2.

Une fois l'installation terminée, déconnectez-vous du serveur ou redémarrez le serveur, le noeud sera automatiquement écrit dans la variable d'environnement.

Entrez git --version pour vous assurer que git est bien , si la commande d'invite de commande git est introuvable, utilisez apt-get install git pour installer git.

Pour accéder au Casnode avec succès, vous devez ouvrir le port 7001 et 8000. :::

Install Golang​

L'utilisateur root exécute les commandes suivantes pour télécharger et décompresser le fichier binaire Go dans le répertoire /usr/local.

wget -c https://dl.google.com/go/go1.16.5.linux-amd64.tar.gz -O - | sudo tar -xz -C /usr/local

Ensuite, nous devons ajouter Golang aux variables d'environnement, sudo vim /etc/profile, Entrez i, ajoutez le code suivant à la dernière ligne du fichier.

export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin

Appuyez sur Échap et entrez :wq pour enregistrer le fichier et quitter.

Ensuite, utilisez la commande source /etc/profile pour faire fonctionner les nouvelles variables d'environnement.

Maintenant, entrez la version go, vous verrez la version go et nous l'avons installée avec succès. Si vous ne pouvez pas vous connecter à GitHub, vous pouvez configurer le miroir. La commande est

go env -w GOPROXY=https://goproxy.cn,direct

Git clone Casnode & Casdoor​

Ensuite, exécutez les commandes suivantes dans le dossier où vous voulez stocker le projet.

git clone https://github.com/casbin/casdoor.git
git clone https://github.com/casbin/casnode.git

Maintenant, vous pouvez voir il ya deux dossiers, Casnode et Casdoor.


Configurer Casdoor​

Lancer Casdoor​

Nous configurons d'abord Casdoor.

cd casdoor
allez construire main.go
vim conf/app.conf

Entrez i, trouver

dataSourceName = root:123@tcp(localhost:3306)/

Changer le mot de passe de MySQL fourni par le panneau BT comme 123, puis appuyez sur Esc, entrez :wq pour enregistrer et quitter.

web cd
npm install
npm run build
cd ..
sudo nohup ./main &

Configurer Casnode dans Casdoor​

Maintenant que Casdoor a été configuré, visitez http://your-ip:8000 pour configurer Casnode.

Le compte de connexion administrateur par défaut est admin/123.

Cliquez sur Organisation, puis sur Ajouter, cliquez sur Modifier pour l'organisation ajoutée, et changez le nom de l'organisation que vous voulez. Ici, je l'ai réglé sur casbin-forum, puis cliquez sur Enregistrer.

Cliquez sur Applications, puis cliquez sur Ajouter, pour l'application que vous venez d'ajouter, cliquez sur Modifier, changer le nom de l'application en nom que vous voulez, je l'ai changé en app-casbin-forum. Cliquez sur l'organisation, sélectionnez l'organisation que vous venez d'ajouter, mon organisation est casbin- forum. Cliquez sur les URL de redirection, modifiez le lien dans la boîte vers http://votre-ip:7000/callback.Enfin, souvenez-vous de l'ID du client et du secret du client, puis cliquez sur Enregistrer.

Cliquez sur Utilisateurs, sur Ajouter, puis sur Modifier, modifier l'utilisateur ajouté, cliquez sur Organisation, sélectionnez le forum de casse et cliquez sur Administrateur. Enfin, cliquez sur Enregistrer, maintenant que votre organisation a un compte administrateur.


Configurer Casnode​

Ensuite, nous configurons dans Casnode.

sudo su
cd casnode
allez construire main.go
vim conf/app.conf

Entrez i et trouvez

dataSourceName = root:123@tcp(localhost:3306)/

Changer le mot de passe MySQL fourni par le panneau BT en 123, puis trouver casdoorEndpoint, le modifier sur http://your-ip:8000 (adresse backend de Casdoor), trouver ClientId et ClientSecret, et les modifier à l'identifiant du client d'application et au secret client précédemment mémorisé, trouver casdoorOrganization, modifier le nom de l'organisation dans votre ensemble. Enregistrer et quitter.

cd web
vim src/Conf.js

Modifier l'URL du serveur à http://your-ip:8000 (adresse d'accès front-end de Casdoor), modifier le ClientId au ClientId de l'application vient d'ajouter, modifier le nom de l'application et modifier l'organisation pour le nom de l'organisation définie. Enregistrer et quitter.

npm install
npm run build
cd ..
nohup ./main &

Prochaine visite http://your-ip:7000, cliquez sur login, entrez le compte que vous avez ajouté avant, user_1/123, vous êtes maintenant connecté avec succès à Casnode.

Pour plus de paramètres, veuillez consulter Casnode.
