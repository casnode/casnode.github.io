在 Linux BT 面板下安装 Casnode​

准备工作​

在安装 BT 面板后，浏览器访问BT 面板，选择软件商店，搜索并安装 MySQL 然后搜索节点，您可以看到有一个 PM2 管理器。安装 PM2 管理器。

安装完成后，从服务器断开连接或重新启动服务器，节点将自动写入环境变量。

输入 git --version 以确保git 是 ， 如果没有找到提示命令 git ，请使用 apt-get 安装 git 来安装git。

要成功访问Casnode ，您需要打开 7001 和 8000 端口。 :::

Install Golang​

根用户执行以下命令以下载并解压Go二进制文件到 /usr/local 目录。

wget -c https://dl.google.com/go/go.16.5.linux-amd64.tar.gz -O - | sudo tar -xz -C /usr/local

然后我们需要将黄金添加到环境变量， sundo vim /etc/profile, 输入 i, 在文件最后一行添加以下代码。

导出 GOROOT=/usr/local/go
导出 PATH=$PATH:$GOROOT/ bin

按 Esc 并输入 :wq 保存文件并退出。

然后使用命令 源 /etc/profile 来使新添加的环境变量发挥作用。

现在输入 转到版本，你会看到转到版本，我们已经成功安装。 如果您不能连接到 GitHub ，您可以设置镜像。 命令是

go env -w GOPROXY=https://goproxy.cn,direct

Git clone Casnode & Casdoor​

接下来，在你想要存储项目的文件夹中执行以下命令。

git clone https://github.com/casbin/casdoog.git
git clone https://github.com/casbin/casnode.git

现在，你可以看到两个文件夹，Casnode 和 Cassdoor。


配置 Cassdoor​

运行卡斯门​

我们首先配置Casdoor。

cd 腰门
去building main.go
vim conf/app.conf

输入 i，找到

dataSourceName = root:123@tcp(localhost:3306)/

更改 BT 面板提供的 MySQL 密码为 123， 然后按 Esc, 输入 :wq 保存并退出。

cd web
npm install
npm building
cd ...
sudo nohup ./main &

配置 Casdoor 中的 Casnode​

既然已配置了 Casdoor ，请访问 http://your-ip:8000 配置Casnode。

默认的管理员登录帐户是 admin/123。

点击组织，然后点击添加，点击编辑添加的组织，并将名称更改为您想要的组织名称。 这里我将它设置为连锁论坛，然后点击保存。

单击应用程序，然后单击添加您刚刚添加的应用程序，单击编辑， 将名称更改为您想要的应用程序名称，我将其更改为应用种姓论坛。 点击组织，选择您刚刚添加的组织，我的组织是卡斯宾论坛。 点击重定向URL，将方框中的链接修改为 http://your-ip:7000/callback。最后，记住客户端 ID 和客户端密钥，然后单击保存。

点击用户，点击添加，然后点击编辑，修改添加的用户，点击组织，选择种姓论坛，并点击管理员。 最后单击保存，现在您的组织有一个管理员帐户。


配置 Casnode​

下一个配置在 Casnode。

sudo su
cd casnode
go building main.go
vim conf/app.conf

输入 i 并找到

dataSourceName = root:123@tcp(localhost:3306)/

将 BT 面板提供的 MySQL 密码更改为 123, 然后找到 casdoorEndpoint, 修改为 http://your-ip:8000 (asdoor 后端地址)， 找到 ClientId 和 ClientSecret, 并将它们修改到以前记住的应用程序客户端ID和客户端秘密，寻找连锁组织，将组织名称修改到您的集合。 保存并退出。

cd web
vim src/Conf.js

修改 serverUrl 到 http://your-ip:8000 (asdoor front-end access address), 将ClientId 修改为刚刚添加的应用程序的 ClientId 修改应用程序名称到设定的应用程序名称，并将组织修改到设定的组织名称。 保存并退出。

npm 安装
npm 运行 build
cd ...
nohup ./main &

下次访问 http://your-ip:7000，点击登录，输入您以前添加的帐户，user_1/123，您现在已成功登录到Casnode。

欲了解更多设置，请参阅 Casnode。
