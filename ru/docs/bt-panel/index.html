Установите Casnode под панель Linux BT​

Подготовка к работе​

После установки панели BT браузер посещает панель BT, выбирает магазин программного обеспечения, ищет и устанавливает MySQL, а затем ищет узел, вы увидите, что есть менеджер PM2, установите менеджер PM2.

После завершения установки отключитесь от сервера или перезапустите сервер, узел будет автоматически записываться в переменную среды.

Введите git --version , чтобы убедиться, что git это , если команды git не найдены, используйте apt-get install git для установки git.

::tip Для успешного доступа к Casnode необходимо открыть порт 7001 и 8000 :::

Install Golang​

Корневой пользователь выполняет следующие команды для загрузки и распаковки бинарного файла Go в каталог /usr/local.

wget -c https://dl.google.com/go/go1.16.5.linux-amd64.tar.gz -O - | sudo tar -xz -C /usr/local

Затем нам нужно добавить Golang в переменные окружения, sudo vim /etc/profile, Enter i, добавьте следующий код в последней строке файла.

экспорт GOROOT=/usr/local/go
экспорт PATH=$PATH:$GOROOT/bin

Нажмите Esc и введите :wq , чтобы сохранить файл и выйти.

Затем используйте команду source /etc/profile , чтобы добавить новые переменные среды.

Теперь введите версию, вы увидите go версию, и мы установили ее успешно. Если вы не можете подключиться к GitHub, вы можете настроить зеркало. Команда

go ruv -w GOPROXY=https://goproxy.cn,direct

Git clone Casnode & Casdoor​

Далее, выполните следующие команды в папке, в которой вы хотите сохранить проект.

git clone https://github.com/casbin/casdoor.git
git clone https://github.com/casbin/casnode.git

Теперь вы можете увидеть две папки: Casnode и Casdoor.


Настроить Casdoor​

Запустить кастрюль​

Мы сначала настраиваем Casdoor.

cd casdoor
перейти к сборке main.go
vim conf/app.conf

Введите i, найти

dataSourceName = root:123@tcp(localhost:3306)/

Изменить пароль MySQL от панели BT как 123, затем нажмите Esc, введите :wq для сохранения и выхода.

cd web
npm install
npm запустить build
cd ..
sudo nohup ./main &

Настроить Casnode в Casdoor​

Теперь, когда Casdoor был настроен, посетите http://your-ip:8000 для настройки Casnode.

Учётная запись администратора по умолчанию — admin/123.

Нажмите на Организацию, затем нажмите Добавить, нажмите Правка для добавленной организации и измените название организации, которую вы хотите. Здесь я установил его в casbin-forum, а затем нажмите кнопку Сохранить.

Нажмите "Приложения", затем нажмите "Добавить" для приложения, которое вы только что добавили, нажмите "Изменить", изменить имя на имя приложения вы хотите, я изменил его на app-casbin-forum. Нажмите на организацию, выберите организацию, которую вы только что добавили, моя организация является казино-форум. Щелкните на Redirect URL, измените ссылку в поле http://your-ip:7000/callback.Finally, запомните ID клиента и Client Secret, и нажмите кнопку Сохранить.

Нажмите на Пользователи, нажмите Добавить, затем нажмите Редактировать, изменить добавленного пользователя, нажмите кнопку Организация, выберите casbin-форум и нажмите кнопку администратора. Наконец нажмите Сохранить, теперь у вашей организации есть учётная запись администратора.


Настроить Casnode​

Далее мы настроим в Casnode.

sudo su
cd casnode
перейти к сборке main.go
vim conf/app.conf

Введите i и найти

dataSourceName = root:123@tcp(localhost:3306)/

Измените пароль MySQL от панели BT на 123, затем найдите casdoorEndpoint, измените его на http://your-ip:8000 (бэкэнд адрес), найти ClientId и ClientSecret, и модифицируйте их к предыдущему идентификатору клиента приложения и секрету клиента, найдите casdoorOrganization, измените название организации к набору. Сохранить и выйти.

cd web
vim src/Conf.js

Изменить serverUrl на http://your-ip: 8000 (адрес внешнего доступа в кассу), изменить ID клиента на Клиента только что добавленного, изменить имя приложения к заданному названию приложения и изменить организацию в установленном названии организации. Сохранить и выйти.

npm установить
npm запустить build
cd ..
nohup ./main &

Следующий визит http://your-ip:7000, нажмите на логин, введите учетную запись, которую вы добавили раньше, user_1/123, вы теперь успешно вошли в Casnode.

Для получения дополнительных настроек смотрите Casnode.
